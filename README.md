# Raízes Negras
 ![Image](https://github.com/user-attachments/assets/f1e90b12-0373-4cd8-a3dc-6b78136e8c00)

Bem-vindo ao projeto **Raízes Negras**, uma plataforma digital interativa dedicada à preservação e valorização da herança cultural afrodescendente no Maranhão.

## Visão Geral

O objetivo do projeto é documentar e promover a rica história e cultura afrodescendente, muitas vezes esquecida ou negligenciada, no estado do Maranhão. A plataforma reúne conteúdos como entrevistas com líderes comunitários, documentos históricos, e materiais audiovisuais para ser um recurso educacional valioso para estudantes, pesquisadores e o público em geral.

## Estrutura do Projeto

### Página Principal

O layout da página é dividido em várias seções:

1. **Cabeçalho (Header):**
   - Contém o logotipo e links de navegação principais, como "Início", "Sobre", "Conteúdo", e "FAQ & Contato".

2. **Seção Hero:**
   - Uma introdução visual ao projeto com uma breve descrição e um mapa do Maranhão.

3. **Seção "O Projeto":**
   - Apresenta os objetivos principais do projeto e destaca a importância da preservação da memória afrodescendente.

4. **Teia Cultural:**
   - Um carrossel interativo que exibe os artigos mais recentes sobre a cultura afro-maranhense. Cada item contém uma imagem, título e link para acessar o artigo completo.

5. **Rodapé (Footer):**
   - Inclui informações institucionais sobre os realizadores e orientadores do projeto, além de logotipos dos parceiros institucionais.

## Tecnologias Utilizadas

- **HTML5**: Para a estruturação do conteúdo da página.
- **CSS3**: Para estilização e layout responsivo.
