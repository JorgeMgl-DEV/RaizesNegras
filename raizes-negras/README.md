# Raízes Negras Project

## Overview
The Raízes Negras project is a web application designed to showcase cultural elements and projects. It features a responsive design with a focus on accessibility and user experience.


## Features
- **Responsive Navbar**: The navbar includes a hamburger menu for mobile devices, allowing users to navigate easily.
- **Cultural Section**: Displays carousel items with text and images related to cultural themes.
- **Project Section**: Highlights various projects with images and descriptions.
- **Footer**: Contains links and information about the project.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd raizes-negras
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```
